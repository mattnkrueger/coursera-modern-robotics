# coursera-modern-robotics

### Description
Course contents of "Modern Robotics" taught at Northwestern University, enrolled via [Coursera](https://www.coursera.org/learn/modernrobotics-course1/home/module/1). 
I took this course as I never took a robotics course in college and wanted a general introduction into the field, techniques, and software used.
